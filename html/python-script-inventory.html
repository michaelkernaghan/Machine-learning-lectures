Overview of Python Scripts for Machine Learning

This page provides an overview of various Python scripts and projects related to machine learning and deep learning. Each project folder contains scripts, datasets, and other resources needed to run the models and understand their functionality.

Project Descriptions

dog-name-model-training

Contains scripts for training and using models to predict dog names. Utilizes GPT-based models and includes data preprocessing and model parameters.

ethereum-transactions

Scripts and datasets for analyzing Ethereum transactions. Includes visualizations and labeled datasets for transaction classification.

examples

Contains example scripts for basic machine learning tasks such as bag-of-words text processing and classification using various datasets.

gas-price-anomaly

Scripts for detecting anomalies in Ethereum gas prices. Includes preprocessing steps and anomaly detection models.

gensim-word-vectors-demo

Demonstrates the use of Gensim for word vector operations including word similarity and analogy tasks using pre-trained GloVe embeddings.

neural-net-transaction-classifier

Contains scripts for building and training neural network models to classify Ethereum transactions. Includes data preprocessing and model evaluation steps.

rules-engine

Scripts for implementing a rules engine to classify transactions based on predefined rules. Includes visualizations for confusion matrices and feature importances.

transformers

Uses transformer models for transaction classification tasks. Includes scripts for data preprocessing, model training, and evaluation.

variational-encoder

Contains scripts for building and training variational autoencoders to detect anomalies in Ethereum transactions. Includes data preprocessing and model evaluation.
