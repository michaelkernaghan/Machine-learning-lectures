Generating Dog Names Using GPT Architecture

This Python script leverages a Generative Pre-trained Transformer (GPT) architecture to generate dog names. The model is trained on a dataset of dog names and can generate new, unique names based on the learned patterns.

Overview

The script performs the following steps:

  • Loads dog names from a CSV file.
  • Creates a dataset for training.
  • Defines a GPT model with transformer layers.
  • Trains the model on the dataset.
  • Generates new dog names using the trained model.

Key Components

1. Data Loading

The script loads dog names from a CSV file and processes them to create a dataset suitable for training the GPT model.

2. Dataset Creation

A custom dataset class is defined to handle the dog names. The class includes methods for tokenizing the names and padding them to ensure uniform length.

3. Model Definition

The GPT model is defined using PyTorch's transformer decoder layers. The model includes embedding layers, positional encoding, and a linear output layer.

4. Training

The model is trained using the cross-entropy loss function and the Adam optimizer. The training loop iterates over the dataset, updating the model's parameters to minimize the loss.

5. Name Generation

After training, the model can generate new dog names by sampling from the learned distribution of characters. The generated names are constructed character by character until an end token is produced or the maximum length is reached.

Script Repository

The full Python script can be found in the GitHub repository: dog-name-GPT.py
